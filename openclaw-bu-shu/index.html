OpenClaw部署

2026-02-01 2 min read # 教程

OpenClaw 是一款运行在自有设备上的个人 AI 助手，所有数据均保存在自己的机器上，同时支持连接 WhatsApp、Telegram、Slack、Discord、Google Chat、Signal、iMessage、Microsoft Teams 等多个平台（国内软件生态还需改善！）。该项目在 GitHub 上拥有超过 135,000 颗星。本文将介绍如何在本地平台（Windows）上部署 OpenClaw。
介绍先到这里，下面开始介绍如何进行部署

前置条件

部署OpenClaw开始前，确保准备好这两样：

  1. 安装Node.js（版本≥22）.

  2. 大模型调用API

    OpenClaw本身不提供AI能力，需要接入外部大模型。本文以OpenRouter为例（它聚合了多个主流模型API）。

安装Node.js

Node的安装十分方便，首先进入网址：Node.js — 下载 Node.js®，点击Winodws 安装程序（.msi）进行下载。

下载完毕后，打开文件一路点next即安装完成

获取OpenRouter API密钥

进入OpenRouter官网，点击Sign Up注册一个账号，然后点击Get Api Key，进入页面，点击Create Api Key创建一个Api Key。

安装OpenClaw

打开命令提示符（cmd），可 win + R 输入 cmd，然后执行以下命令：

npm install -g openclaw@latest

openclaw onboard --install-daemon

这时会进入交互式配置界面，根据提示一步步设置：


配置过程中有几个关键点需要注意：

  1. 在出现 Model/auth provider的时候，选择我们注册的 OpenRouter，随后输入我们在OpenRouter中创建的 API key

  2. 在调用模型的选择上，如果OpenRouter账户没有绑定支付方式，务必选择带有“free”字样的模型。

推荐选择： openrouter/tngtech/tng-r1t-chimera:free

后续的skills等配置就可以按照自己的需要进行选择了。

启动并体验

在完成配置后，输入：

openclaw gateway --port 18789 --verbose

然后打开浏览器访问：http://127.0.0.1:18789

你会看到 ClawdBot 的 Web 控制台，接下来就可以愉快的开启你的OpenClaw之旅了~
